# RYM Grenergy - Redesigned Website

A modern, interactive redesign of the RYM Green Energy website built with React, JavaScript, and Framer Motion, featuring a sleek black and green theme.

## 🚀 Features

- ⚡ Built with React 18 and Vite for blazing-fast performance
- 🎨 Stunning black and green theme
- ✨ Smooth animations powered by Framer Motion
- 📱 Fully responsive design
- 🎯 Modern UI/UX with interactive elements
- ♿ Accessible and SEO-friendly

## 🛠️ Technologies

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Framer Motion** - Animation library
- **React Icons** - Icon components
- **CSS3** - Modern styling

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌐 Development

The development server runs on `http://localhost:5173`
